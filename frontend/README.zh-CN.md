# logistics-frontend

🚚 物流管理系统 · 前端项目

本项目是 **logistics-example** 的前端部分，基于 **Vue 3 + Element Plus** 构建，  
与 Spring Boot 后端配合，实现了 **JWT 登录认证、基于角色的动态菜单、订单管理与统计可视化**。

---

## 🧱 技术栈

- **前端框架**：Vue 3（Composition API）
- **构建工具**：Vite
- **UI 框架**：Element Plus
- **请求库**：Axios
- **图表库**：ECharts
- **路由管理**：Vue Router
- **认证方式**：JWT + LocalStorage
- **日志系统**：自定义前端日志模块

## 🔐 登录与权限控制

### 登录流程

- 用户输入用户名和密码
- 后端返回：
    - JWT Token
    - 用户信息（id、用户名、角色）
- Token 存储在 `localStorage`

### 权限控制

- 根据角色动态生成菜单
- 路由守卫限制页面访问
- Axios 请求自动携带 JWT

---

## 👤 角色与权限说明

| 角色 | 权限说明 |
|-----|---------|
| USER | 查看 / 创建自己的订单，查看自己的统计 |
| ADMIN_L2 | 管理被指派订单，查看自己的统计 |
| ADMIN_L1 | 管理全部订单、指派任务、查看全部统计 |

前端界面会根据角色自动变化。

---

## 📝 注册规则说明

- 用户名 / 密码：
    - 自动去除首尾空格
    - 中间禁止包含空格
- 前端不提供角色选择
- 管理员注册需输入管理员密码
    - 密码错误 → 自动注册为普通用户

## 📊 订单统计模块

- 本周
- 本月
- 本年
- 历史统计

展示方式：

- 折线图（ECharts）
- 表格数据

同一接口，根据角色返回不同统计结果。
